# Deutsch Tools

Шпаргалка и тренажёр по немецкому языку (A1 → A2).

🌐 [r4-studio.github.io/deutsch-tools](https://r4-studio.github.io/deutsch-tools)

## Структура

- `index.html` — главная страница с двумя кнопками
- `cheatsheet.html` — шпаргалка: словарь, грамматика, правила, поиск
- `trainer.html` — тренажёр с карточками и тестами (mc, tiles, fill, conj)
- `data.js` — единый источник данных (752 слова, 100 вопросов)

## Локальный запуск

Двойной клик по `index.html`. Никакого билда не нужно — React и Babel
подгружаются с CDN.

## Этапы

- ✅ Шпаргалка и тренажёр работают раздельно
- 🔄 Объединение данных через `data.js` (в процессе)
- ⏳ Триграммный поиск
- ⏳ Прогресс пользователя
