## Шаги завтра

### 1. Положить файлы в репо (5 минут)
- Скачать 3 обновлённых файла (data.js, cheatsheet.html, trainer.html)
- Положить в `D:\r4-GitHub\deutsch-tools\` (перезаписать старые)
- Открыть `index.html` локально

### 2. Локальная проверка шпоры (~15 минут)
- Открыть `cheatsheet.html`
- Открыть DevTools → Console — должно быть `✓ Шпора отрендерена из data.js`
- Пройтись глазами по табам:
  - 🆕 Новые — слова с `new: true`
  - 🔢 Числа — vocab pos=num
  - 🔤 Звуки — SOUNDS
  - ⚡ Глаголы — vocab pos=verb по topics verbs:*
  - 🔥 Unregel — CONJUGATIONS
  - 📦 Существительные — vocab pos=noun по topics nouns:*
  - 🎨 Прилагательные, 🔄 Наречия, 👤 Местоимения
  - 📋 Термины, 📐 Правила, 💬 Фразы
- **Если что-то не на месте** — пиши в чат, чиним
- **Если ок** — заметить что отвалилось визуально (стили, эмодзи, поведение поиска)

### 3. Локальная проверка тренажёра (~15 минут)
- Открыть `trainer.html` — должен загрузиться через 1-2 сек (CDN React+Babel)
- Перейти Глаголы → Modalverben → Тест
- Должно быть до 30 вопросов разных типов (mc/fill/conj/tiles)
- На карточке должны быть пометки «Уровень N · type · авто» где применимо
- Проверить FlipCard (Учить) на любом сабблоке
- «Тест по всему блоку» — должен суммировать все сабблоки

### 4. Коммит (5 минут)
- GitHub Desktop → Summary: `feat: data.js as single source for cheatsheet and trainer`
- Commit to main → Push origin
- Подождать 1-2 мин — GitHub Pages обновится
- Проверить https://r4-studio.github.io/deutsch-tools/

### 5. Косметика и баги (по фактическим находкам)
- Что выловил визуально — фиксим точечно
- Если есть свежий v32 шпоры — перегенерим data.js скриптом за минуту

### 6. В группу (после уверенности)
- Кинуть ссылку в чат
- Собирать фидбек

## Точки возврата

Каждый из 3 файлов независим:

- **Если шпора сломалась** — можно временно убрать `<script src="data.js">` и блок рендерера → старая разметка вернётся (она в файле осталась как fallback).
- **Если тренажёр сломался** — можно временно вернуть на старую версию через git revert.
- **Если data.js сломал что-то в обоих** — git reset на предыдущий коммит вернёт всё.
